# ユーザーフロー

1. **ユーザー登録**
   1. Google認証
2. **旅行プロジェクトの作成**
   1. 入力
      1. タイトル（must）
      2. 参加者（1 or 複数）
      3. 開始日（must）
      4. メモ（任意）
   2. 出力
      1. 旅行プロジェクト招待リンク
3. **旅行プロジェクト招待リンクを共有**
   1. LINE等に共有
4. **旅行開始ボタンを押す**
   1. マップ上に経路が表示される
5. **通知が来る**
6. **写真を撮る**
   1. 撮影した場所にピン📍が刺され写真が表示される
   2. 写真表示
7. **旅行終了ボタンを押す**
8. **旅行の記録が地図上に表示される**
